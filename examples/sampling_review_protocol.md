# Sampling Review Protocol

## Purpose

本文件用于说明第二阶段公开数据集质量验证中的标注质量抽样复核规则。

自动化清洗主要解决文件级、格式级、基础质量级和工程合规分流问题，不能完全替代人工对标签准确性、标签映射合理性和风险语义的判断。因此，进入候选流程的数据集需要进行人工抽样复核。

## Sampling Scope

抽样对象为已经完成格式标准化、通过初步自动化清洗，并进入候选流程的数据集。

因许可限制、授权状态不满足当前公开仓库要求、严重文件损坏或大规模不可解析而未进入候选流程的数据集，不纳入当前轮次的标注质量抽样范围。此类数据集记录为 isolated_license、metadata_only 或其他相应状态，而不是记录为抽样数量为 0。

## Sampling Ratio

当前阶段原则上对每个进入候选流程的数据集随机抽取 5% 样本开展人工复核。

当前批次中，HC3 进入候选流程，样本数量为 400，抽样比例为 5%，对应抽样数量为 20。

## Review Items

每个抽样样本至少复核以下内容：

1. 原始标签是否与样本内容一致；
2. 映射后的统一标签是否合理；
3. AIGC 真实性标签是否合理；
4. 风险类型标签是否与内容匹配；
5. 是否存在多义性、边界样本或无法判断情况；
6. 若标签错误，记录错误类型和修正建议。

## Accuracy Calculation

标注准确率计算公式：

标注准确率 = 抽样复核中标签正确的样本数 / 抽样总样本数 × 100%

当前阶段将 90% 标注准确率作为数据集级准入参考阈值。

## Common Error Types

常见标注错误类型包括：

- 原始标签与样本实际内容不一致；
- 数据集内部标签边界定义不统一；
- 原始多分类标签映射到统一标签体系时发生偏差；
- 样本内容存在多义性或难以判断；
- 元数据关联错误导致样本与标签错配。
